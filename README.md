# Audio-Based Intelligent Traffic Management System for Smart Cities

An AI-powered traffic management system that detects emergency vehicle sirens using acoustic signal analysis and deep learning models.

## Features
- Emergency siren detection
- Audio preprocessing and MFCC feature extraction
- Real-time prediction workflow
- Intelligent traffic signal prioritization
- Django-based monitoring interface

## Tech Stack
- Python
- TensorFlow / Keras
- Librosa
- NumPy
- Scikit-learn
- Django

## Models Used
- MLP
- DNN
- LSTM

## Objective
To reduce emergency vehicle delays and improve traffic flow using AI-driven acoustic analysis instead of traditional camera-only systems.

## Future Improvements
- Edge deployment using TensorFlow Lite
- IoT-enabled smart traffic integration
- Multi-junction coordination
